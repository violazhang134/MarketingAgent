# Marketing Agent: The AI-Publisher 🚀

> **架构即认知，文档即记忆，变更即进化。**

Marketing Agent 是一个专为全栈开发者/小型团队设计的“自动驾驶”发行系统。它不仅仅是一个工具集，而是一个具备感知、决策和行动能力的 AI 发行中台。

## 核心哲学：AI-Publisher

我们拒绝平庸的“配置化工具”。Marketing Agent 的核心承诺是：**你负责创造，我负责发行。**

通过 **观察 (Observe)**、**决策 (Decide)**、**行动 (Act)** 和 **学习 (Learn)** 的闭环，我们致力于让任何作品在 30 分钟内完成首次发布，并在 72 小时内获得数据驱动的迭代建议。

## 四大支柱 (The 4 Pillars)

1.  **Distribution-ready Pack (一键发布包)**: 自动生成高转化封面、分镜脚本、广告文案及落地页，直接对标线上的成熟素材。
2.  **Experiment Runner (全自动实验)**: 强制标准化的 A/B 测试。Agent 自动配置对照组、流量路由及判定规则。
3.  **Telemetry & Attribution (深度数据回环)**: 内置标准化的 Growth 事件（落地页开启、有效开玩、转化归因），无需复杂埋点。
4.  **Next Best Action (下一步最佳动作)**: 拒绝虚无的报表，直接通过规则引擎告诉用户：“关停素材 A，因为素材 B 在巴西地区表现更优”。

## 极速开始

